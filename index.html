Project Overview

Business Problem & Objectives

In e-commerce, not every sale is profitable. Heavy discounts, high shipping costs, and low-margin product categories can erode or eliminate profits — even on large revenue orders. This project builds a machine learning classifier to predict order profitability before fulfillment, enabling Amazon to intervene with smarter pricing and discount strategies.

Research Questions

Can we predict whether an order will be profitable? Which categories and features drive losses most?

Target Variable

Binary classification — Profitable (1) vs Loss (0) based on order profit margin.

Approach

Full ML pipeline: EDA → Preprocessing → Model training → Evaluation → Business insights.

Impact

Real-time profitability scoring at checkout to flag high-risk orders before resources are committed.

Methodology

Data Science Pipeline

1

Exploratory Data Analysis

Analyzed distributions, correlations, and patterns across 16 features. Generated 10 meaningful visualizations including heatmaps, boxplots, and category-level loss rate charts.

2

Data Preprocessing

Encoded categorical variables (Category, State, Sales Size) using LabelEncoder. Applied StandardScaler for Logistic Regression. Stratified 80/20 train-test split.

3

Model Development

Trained and compared 3 models: Logistic Regression (baseline), Random Forest, and Gradient Boosting. Used 8 features including Sales, Quantity, Category, State, and Shipping Days.

4

Model Evaluation

Evaluated using Accuracy, F1-Score, and ROC-AUC. Generated confusion matrices and ROC curves for all models. Selected Gradient Boosting as the best performer.

5

Business Insights

Translated technical findings into actionable recommendations for pricing strategy, category management, and real-time order screening.

Results

Model Performance Comparison

Gradient Boosting outperforms both alternatives across all three metrics, achieving the highest discriminative power with a ROC-AUC of 0.923.

💰

Reprice furniture categories

Audit discount policies for Tables, Bookcases, and Chairs. Set minimum price floors or renegotiate supplier costs to reduce 130+ annual loss-generating orders.

📈

Expand zero-loss category offerings

Paper, Art, and Appliances have 0% loss rate over 4+ years. Increasing inventory depth in these categories offers reliable, low-risk revenue growth.

🤖

Deploy real-time profitability scoring

The Gradient Boosting model (AUC 0.923) can be deployed as an API to flag high-risk orders at checkout, enabling price adjustments before fulfillment.

📍

Review Colorado & Washington markets

These states show above-average loss rates. A region-specific pricing review can identify whether the issue is logistics costs or aggressive discounting.

📋

Implement discount approval workflows

Sales Amount is the 2nd most important predictor. Large orders that still generate losses likely have excessive discounts — approval workflows can prevent many losses.
